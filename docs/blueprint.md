# **App Name**: FlowState

## Core Features:

- Session Timer: Session timer with start, pause, break, resume, and end functions.
- Session Categorization: Ability to categorize sessions (coding, learning, etc.) with titles and notes.
- Session Metrics: Visual display of session duration, break times, and category splits.
- Streak Counter: Track and display focus streak to motivate continued use.
- AI Focus Tool: Tool to provide AI-driven suggestions for optimal focus times based on user data.

## Style Guidelines:

- Primary color: Soft blue (#77B5FE) for calm focus and productivity.
- Background color: Light gray (#F0F4F8) to minimize distractions.
- Accent color: Muted green (#82E0AA) to highlight achievements and milestones.
- Font pairing: 'Inter' for body text (sans-serif) and 'Space Grotesk' for headings (sans-serif) to maintain a modern and readable interface.
- Simple, clean icons representing session categories and actions.
- Dashboard layout with key stats at the top and session timer in the center for easy access.
- Subtle animations for session start/stop and streak achievements.
